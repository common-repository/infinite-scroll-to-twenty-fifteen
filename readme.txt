== Installation ==

### Installation

1. Upload 'infinite-scroll-to-twenty-fifteen' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

### Usage

* Just activate the plugin to add Infinite Scroll on Twenty Fifteen theme.

### Change Text Messages

1. Go to your wp-admin > Settings > Reading.
2. Go to Infinite Scroll Text Messages.
